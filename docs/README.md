## ドキュメント更新

ドキュメントは実装と同期して更新する必要があります：

1. **機能実装時**: 対応するAPIドキュメントを更新
2. **バグ修正時**: トラブルシューティングガイドを更新  
3. **設定変更時**: セットアップガイドを更新
4. **新機能追加時**: ユーザーガイドを追加
